# Online Job Portal System

## Description
The Online Job Portal System is a web-based application that connects job seekers and employers on a single platform. Job seekers can create profiles, upload resumes, search for jobs, and apply online, while employers can post job vacancies, manage applications, and shortlist candidates.

## Features
- User Registration and Login
- Job Search and Filtering
- Resume Upload
- Job Application Management
- Employer Dashboard
- Admin Panel
- Application Tracking

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap
- Java/Spring Boot
- MySQL

## Objective
To simplify the recruitment process by providing an efficient platform for employers and job seekers.
